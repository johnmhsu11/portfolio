# Static Dashboard Screenshots

Drop a PNG screenshot for each dashboard here, named exactly after its `id` in `dashboards.js`:

| File                    | Dashboard                         |
|-------------------------|-----------------------------------|
| `us-cost-of-living.png` | US Cost of Living                 |
| `life-expectancy.png`   | Life Expectancy by Nation         |
| `auto-value.png`        | Automobile True Value vs. Price   |

## How to capture a screenshot

1. Open the live dashboard (click "Open ↗" on the embed page)
2. Wait for it to fully load
3. Take a full-page screenshot (browser zoom: 80-90% works well)
4. Save as a PNG with the exact filename above and drop it in this folder
5. Commit and push — Vercel will pick it up automatically
